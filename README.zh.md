## 📸 使用效果

### 📱 程序坞启动
点击程序坞图标，自动在 `文稿/Claude Code` 文件夹启动

### ⚡ Finder工具栏启动  
在任意文件夹点击工具栏图标，直接在当前文件夹启动

### 🚀 零配置使用
无需任何设置，安装即用，自动跳过权限确认

### 💡 智能终端选择
**首次启动时的贴心功能**：
- 如果系统有多个终端（iTerm2、Warp、Terminal等），会自动弹出选择框
- 选择后会记住你的偏好，下次直接使用
- 支持的终端：iTerm2、Warp、Alacritty、Terminal
- 随时可以通过 `macos/set-terminal.sh` 更改偏好
